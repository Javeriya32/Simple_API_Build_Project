# Simple User API

## Overview
This project is a simple ASP.NET Core Web API that provides CRUD operations for managing users.

## Copilot Assistance Summary
Microsoft Copilot assisted in generating the API structure, CRUD endpoints, validation logic, and middleware implementation. It also helped debug endpoint logic, improve request handling, and ensure best practices for minimal APIs and middleware usage.

## Features
- CRUD operations (GET, POST, PUT, DELETE)
- Input validation using data annotations
- Custom request logging middleware
- In-memory user storage

## Technologies
- ASP.NET Core Minimal API
- .NET 8
